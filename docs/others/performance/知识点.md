# 知识点

## 布局抖动

布局抖动是指`DOM`元素被`JavaScript`多次反复读写，导致文档多次无意义重排。我们知道浏览器很“懒”，它会合并（`batch`）当前操作，统一进行重排。可是，如果在当前操作完成前从`DOM`元素中获取值，那么就会迫使浏览器提早执行布局操作，这被称为强制同步布局。这样做对于低配置的移动设备来说，后果是不堪设想的。
