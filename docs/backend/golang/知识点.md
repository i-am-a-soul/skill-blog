## `go`语句

```go

import "fmt"

func main () {
	for i := 0; i < 10; i++ {
		go func () {
			fmt.Println(i)
		}()
	}
}

// 控制台什么都没有输出
```

## `range`

```go
package main

import "fmt"

func main () {
	arr := [...]int{1, 2, 3, 4, 5, 6}
	maxIdx := len(arr) - 1
	for idx, ele := range arr {
		if idx == maxIdx {
			arr[0] += ele
		} else {
			arr[idx + 1] += ele
		}
	}
	fmt.Println(arr) // [7 3 5 7 8 11] 而不是 [22 3 6 10 15 21]
}
```
