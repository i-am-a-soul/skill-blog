# 知识点

## 服务端渲染

服务端渲染是指由服务端完成页面`HTML`的渲染，然后发送到客户端，再由客户端为其绑定状态与事件，使之成为完全可交互页面的一种技术。

服务端渲染通常可以带来更好的用户体验，并且对于那些**首屏渲染时间与转化率直接相关**的应用程序而言，服务端渲染至关重要。
