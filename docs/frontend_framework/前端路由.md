# 前端路由

![](/skill-blog/img/0055.png)

| 特点               | Hash路由      | Browser路由            |
| ------------------ | ------------- | ---------------------- |
| 美观度             | 不好，有`#`号 | 好                     |
| 易用性             | 简单          | 中等，需要`Server`配合 |
| 依赖`Server`端配置 | 不依赖        | 依赖                   |
| 跟锚点功能冲突     | 冲突          | 不冲突                 |
| 兼容性             | `IE8`         | `IE10`                 |
